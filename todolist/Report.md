
# App 01 – To-Do List App

## Description

A modern **To-Do List** mobile application developed using **React Native** and **Expo Snack**.
The app enables users to efficiently manage daily tasks by allowing them to **add, edit, delete, and update goals** through a clean, intuitive, dark-themed interface.

The design focuses on usability, clarity, and providing a professional user experience inspired by modern productivity applications.

---

## Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Dynamic Add / Update button
* Scrollable task list using FlatList
* Keyboard handling using KeyboardAvoidingView
* Input validation using alert messages
* Modern dark UI with 3D card-style design
* Responsive and clean layout

## Color Palette

The application uses a dark UI color system inspired by productivity and task management apps:

Main Background: Dark Navy – #0F172A
Card Background: Dark Slate – #1E293B
Primary Accent: Neon Blue – #38BDF8
Update Action: Emerald Green – #10B981
Delete Action: Red – #EF4444
Borders: Muted Blue-Gray – #334155
Primary Text: Off White – #F8FAFC
Secondary Text: Soft Gray – #CBD5E1
Placeholder Text: Light Gray – #94A3B8

The dark theme improves focus, reduces eye strain, and gives the application a modern professional feel.
